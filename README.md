# Trick of the Day

## Axle Drop-In

An axle stall is a stall on both trucks of a skateboard. It is used commonly to regain composure before performing another trick or to “drop in” on a ramp. Essentially a stationary 50-50.

**Invented by:** Unknown

**Year:** Unknown
