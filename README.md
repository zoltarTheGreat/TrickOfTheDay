# Trick of the Day

## Rock to Fakie

Similar to the Rock to Fakie only a quick 180 is done as you come off the lip so that you don’t ride fakie. The frontside variation is much harder and is considered one of the most stylish lip tricks.

**Invented by:** Unknown

**Year:** Unknown
