# Trick of the Day

## Lien Air

Another of the basic airs. It’s a frontside air grabbing the nose or heel edge with your front hand (leading hand). Neil Blender invented the Lien Air. Lien is the creator’s name spelled backwards.

**Invented by:** Unknown

**Year:** Unknown
