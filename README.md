## Noseslide

A noseslide is performed by riding parallel to an obstacle (ledge, rail, etcâ€¦) The skateboarder then does an ollie and turns the board 90 degrees. They then land on the ledge with the nose of the board sliding on top of it. This can be done frontside or backside. The skateboarder can then come off the ledge either regular or fakie (backwards).
