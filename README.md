# Trick of the Day

## Frontside Salad

An ollie in which both the skater and the skateboard rotate 180 degrees either frontside or backside. If the skater spins some other multiple of 180 degrees (e.g. 360, 540), the trick is named accordingly.

**Invented by:** Unknown

**Year:** Unknown
